Portal queries
**************

I see no active sessions in My Interactive Sessions?
====================================================

Some accounts find that after clicking the `Launch` button, they see “Session was successfully created”, but there are still no active sessions in My Interactive Sessions even after refreshing the page, restart the browser, and trying many times.


There is an issue with setting a default shell to zsh. This is a known issue with the current OOD setup. A temporary workaround is to change the default shell to bash.
